# Política de Privacidad

**Última actualización: 31 de diciembre de 2025**

## Resumen

User Agent Changer es una extensión de navegador diseñada con la privacidad como su principio fundamental. Esta política de privacidad explica nuestras prácticas de datos en términos claros: **no recopilamos nada, no almacenamos nada y no compartimos nada**.

## Recopilación de Datos

**NO recopilamos NINGÚN dato en absoluto.**

Esta extensión:
- NO recopila ninguna información personal
- NO recopila ningún historial de navegación
- NO recopila ninguna URL de sitios web que visites
- NO recopila estadísticas de uso ni análisis
- NO recopila ninguna información del dispositivo
- NO te rastrea de ninguna manera

## Almacenamiento de Datos

**NO almacenamos NINGÚN dato en absoluto.**

Esta extensión:
- NO almacena ningún dato en servidores remotos (porque no tenemos ninguno)
- NO crea cuentas de usuario ni perfiles
- NO mantiene ninguna base de datos con información de usuarios
- NO guarda registros de tu actividad
- NO guarda tus patrones de navegación

### Almacenamiento Local

La extensión solo utiliza el almacenamiento local de tu navegador para guardar:
- Tus preferencias de user agent (almacenadas localmente SOLO en TU dispositivo)
- Tu configuración de spoofs permanentes personalizados (almacenada localmente SOLO en TU dispositivo)

Estos datos:
- Nunca salen de tu dispositivo
- Nunca se transmiten a ningún lugar
- Nunca se comparten con nadie
- Solo son accesibles por la extensión en tu navegador local
- Se pueden eliminar en cualquier momento desinstalando la extensión

## Compartición de Datos

**NO compartimos NINGÚN dato en absoluto.**

Esta extensión:
- NO envía datos a terceros
- NO se comunica con ningún servidor externo
- NO tiene ninguna infraestructura backend
- NO se integra con ningún servicio de análisis
- NO comparte datos con anunciantes (no tenemos anuncios)
- NO vende tus datos (no tenemos datos para vender)

## Cómo Funciona la Extensión

La extensión User Agent Changer opera completamente de forma local en tu dispositivo:

1. **Modificación del User Agent**: Cuando navegas por la web, la extensión intercepta las peticiones del navegador y modifica la cadena del user agent antes de que la petición se envíe. Esto sucede completamente dentro de tu navegador.

2. **Verificación de URL**: La extensión comprueba la URL de la página actual contra tu lista de spoofs permanentes almacenada localmente para determinar si se debe aplicar un user agent personalizado. Esta verificación ocurre localmente en tu dispositivo.

3. **Sin Comunicación Externa**: La extensión NO envía las URLs que visitas, los user agents que usas, ni ninguna otra información a ningún servidor externo.

## Servicios de Terceros

Esta extensión NO utiliza ningún servicio de terceros, incluyendo:
- Ningún servicio de análisis
- Ningún servicio de reporte de errores
- Ninguna red publicitaria
- Ningún píxel de seguimiento
- Ninguna API externa
- Ninguna red de distribución de contenido para recopilación de datos

## Permisos Explicados

La extensión solicita ciertos permisos del navegador para funcionar:

- **webRequest/webRequestBlocking**: Para interceptar y modificar el encabezado del user agent antes de que se envíen las peticiones
- **Storage**: Para guardar tus preferencias localmente en tu dispositivo
- **Tabs**: Para detectar qué URL está activa y aplicar el user agent apropiado
- **Todas las URLs (<all_urls>)**: Para poder modificar el user agent en cualquier sitio web que visites

Estos permisos se utilizan únicamente para la funcionalidad principal de la extensión y NO para la recopilación de datos.

## Privacidad de Menores

Dado que no recopilamos ningún dato en absoluto, esta extensión es segura para usuarios de todas las edades. No recopilamos conscientemente ninguna información de nadie, incluidos menores de 13 años.

## Cambios en Esta Política

Aunque nuestro compromiso con la privacidad nunca cambiará, podemos actualizar esta política para reflejar:
- Cambios en los requisitos de las extensiones de navegador
- Aclaraciones solicitadas por los usuarios
- Actualizaciones de requisitos legales

Cualquier cambio se publicará en esta página con una fecha de "Última actualización" actualizada.

## Tus Derechos

Dado que no recopilamos ni almacenamos ningún dato sobre ti:
- No hay datos a los que acceder
- No hay datos que eliminar
- No hay datos que exportar
- No hay datos que corregir

Sin embargo, mantienes el control completo sobre tu configuración local y puedes:
- Ver tus preferencias locales a través de la página de opciones de la extensión
- Modificar tus preferencias en cualquier momento
- Eliminar todos los datos locales desinstalando la extensión

## Código Abierto

Esta extensión es de código abierto, lo que significa que cualquiera puede revisar el código para verificar que hacemos exactamente lo que afirmamos: absolutamente nada con tus datos.

## Contacto

Si tienes preguntas sobre esta política de privacidad o las prácticas de datos de la extensión (o la falta de ellas), puedes:
- Revisar el código fuente para verificar nuestras afirmaciones
- Abrir un issue en nuestro repositorio
- Contactarnos a través de los canales de soporte de la extensión

## Resumen Final

**Somos agnósticos a tus datos. No recopilamos nada, no almacenamos nada remotamente y no compartimos nada. La extensión opera completamente de forma local en tu dispositivo, simplemente modificando la cadena del user agent para tus sesiones de navegación. Eso es todo. Nada más.**

Tu privacidad no es una característica—es nuestra base fundamental.
